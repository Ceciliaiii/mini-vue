# 靶向更新

## 模板
```html
<div :class="activeClass">
  <span>静态文本</span>
  {{ dynamicText }}
</div>
```

## 编译生成的 render
```ts
function render() {
  openBlock(); // 开启动态子节点收集
  // 创建带 PatchFlag 和 dynamicChildren 的 vnode
  return createElementBlock(
    'div',
    { class: activeClass, patchFlag: PatchFlags.CLASS }, // PatchFlag 标记动态 class
    [
      createVnode('span', null, '静态文本'), // 静态子节点（不加入 dynamicChildren）
      createVnode(Text, null, dynamicText, PatchFlags.TEXT) // 动态文本（加入 dynamicChildren）
    ]
  );
}
```
