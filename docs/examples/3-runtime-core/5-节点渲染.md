# Text & Fragment 节点渲染

## `vnode.type === Text`
```ts
// 初始化
render(h(Text, "czx"), app)


// 重渲染
setTimeout(() => {
    render(h(Text, "czx handsome"), app)
}, 1000)
```



## `vnode.type === Fragment`

```ts
render(h(Fragment, [h('div', "hello"), h('a', "world")]), app)

// 修改a标签的文本
setTimeout(() => {
    render(h(Fragment, [h('div', "hello"), h('a', "hello")]), app)
}, 1000)
```
